📚 童話故事屋

給孩子的有聲故事 App

關於這個 App

童話故事屋收錄多篇經典中文童話，由真人語音朗讀，搭配手繪風插圖，讓孩子在睡前或任何時候都能輕鬆享受故事時光。

常見問題

  • 如何恢復購買？開啟 App → 設定 → 訂閱管理 → 點選「恢復購買」。
  • 如何取消訂閱？前往 iOS 設定 → Apple ID → 訂閱 → 選擇「童話故事屋」→ 取消訂閱。
  • 試用期結束後會自動扣款嗎？是的，請在試用到期前 24 小時取消，才不會被扣款。
  • 語音品質不佳？進入 iOS 設定 → 輔助使用 → 語音 → 朗讀內容，將語音品質改為「增強」。

刪除帳號與資料

童話故事屋不需要帳號即可使用，App 本身不收集任何個人識別資訊。
